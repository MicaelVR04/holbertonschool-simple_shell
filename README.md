## Title

- **Unix-Simple_Shell**: This project is a replicate of a Unix-Shell.

## How it works
- **Prototype**: A Unix shell is the program that reads what you type in the terminal and makes it happen.
- **Description**: This project is custom implementation that aims to replicate the functionality of the Unix-Simple_Shell in C.
